# freecodecamp-semantic-html-practice
A collection of Semantic HTML practice projects built while learning the FreeCodeCamp Responsive Web Design certification

## Workshop Build Major Browsers List
- In this workshop, build a list of major web browsers.
## Workshop Build Quincy Job Tips Page
- In this workshop, you will practice working with semantic HTML by building a web page that includes some of Quincy Larson's tips for landing a developer job.
## Workshop Build a Cat Blog Page
- In this workshop, you will practice working with semantic HTML by building a blog page dedicated to Mr. Whiskers the cat.
